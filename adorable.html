SUMMARY

Building things people love

My main project at Adorable was developing major features for a real estate blogging and SEO application. The main initiative was to focus on user engagement and generate more traffic. The project was a Ruby on Rails application with JavaScript frameworks including jQuery and Knockout.js. Overall, the project suffered from severe amounts of technical debt. Here are some of the major accomplishments during my time on the project:

  • Delivered major features designed to improve the user experience
  • Refactored complex database queries which saw more than 50% improvement in load times
  • Resolved issues with developer environment setup (migrations/seeding)
  • Introduced TDD and a test framework to the project, reducing regressions
  • Implemented scripts to automate deployments, saving 5+ hours per month

Overall, our efforts on the project resulted in increased user excitement and an improved experience when browsing real estate agent blog posts.
